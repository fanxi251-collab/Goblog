



### Go动态博客系统项目（CMS内容管理）介绍

首页布局为顶部导航菜单和下方内容。
导航栏菜单右侧对齐写博客系统的名字：奇迹之夏

导航菜单横向放置：

①首页：放置博客的介绍信息、我的一些个人介绍（采用滚动式分页逻辑）

②专栏：专栏下有三个栏目：技术、生活、游戏，后续可能会加入新增栏目功能（CSS Grid实现文章的卡片式布局）

③留言板：支持访客评论互动

文章的编辑和发布会有一个独立后台，通过隐秘路径访问（参考word**WordPress**、**Typecho**、**Halo**）

前端界面风格：清新活力优雅

### 技术栈

| 模块         | 技术                        | 说明                 |
| :----------- | :-------------------------- | :------------------- |
| Web框架      | Gin                         | 轻量高性能           |
| 数据库       | SQLite                      | 初期足够             |
| ORM          | GORM                        | 带 Repository 层隔离 |
| 模板引擎     | Jet                         | 性能好、支持继承     |
| 会话管理     | gin-contrib/sessions        | cookie存储           |
| 密码加密     | bcrypt                      |                      |
| 配置管理     | Viper                       | 支持 yaml            |
| Markdown渲染 | goldmark                    |                      |
| XSS清洗      | bluemonday                  | 清洗评论+文章HTML    |
| 文件上传接口 | 自定义 FileService          | 本地实现，预留扩展   |
| 后台路径安全 | 配置化 + HTTP Basic Auth    | 双重保护             |
| 数据库迁移   | GORM AutoMigrate + 备份提醒 | 初期可用             |

### 前端技术栈

| 用途       | 技术                      | 说明                       |
| :--------- | :------------------------ | :------------------------- |
| 样式       | Tailwind CSS 或 自定义CSS | 快速实现清新活力风格       |
| 布局       | CSS Grid + Flexbox        | 卡片网格、响应式           |
| 后台编辑器 | Vditor                    | 轻量、中文友好             |
| 评论交互   | 原生表单 + AJAX（可选）   | 初期可直接提交刷新         |
| 无限滚动   | 传统分页按钮（简单）      | 后续可升级为 htmx 或 fetch |
